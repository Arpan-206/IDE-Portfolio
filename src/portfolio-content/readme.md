# SSH Portfolio

The best way to see my work isn't this website — it's your terminal.

```bash
ssh arpanpandey.dev
```

No password. No keys. Just press enter.

Built with [Charm's Bubble Tea](https://github.com/charmbracelet/bubbletea)
for the TUI and [Wish](https://github.com/charmbracelet/wish) for the SSH server.

## Why?

Because portfolios don't have to live in browsers.
Inspired by terminal.shop — made it my own with Bubble Tea's component model.
